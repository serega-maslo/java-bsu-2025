Данный проект представляет собой веб-игру «Крестики-нолики» с системой учёта рекордов. Основная цель работы — создание полноценного стека технологий (Full-stack) и его развертывание с помощью Docker.
Что было сделано:
1. Бэкенд (Spring Boot)

    Разработан REST API для обработки результатов игр.

    Создана модель данных Player для хранения имен игроков и количества их побед.

    Использован Spring Data JPA для работы с базой данных (автоматическое сохранение и получение данных).

    Настроен контроллер, который записывает победу по завершении игры и выдает топ игроков.

2. Фронтенд (HTML5/JS)

    Сверстано игровое поле на CSS Grid.

    Написана логика игры на чистом JavaScript: переключение ходов, определение победителя и ничьей.

    Реализовано взаимодействие с сервером через Fetch API (обновление статистики без перезагрузки страницы).

    Добавлен автофокус на поле ввода имени первого игрока при загрузке страницы.

3. База данных (PostgreSQL)

    Интегрирована реляционная база данных PostgreSQL для хранения статистики.

    Настроена автоматическая инициализация таблиц при запуске приложения.

4. Контейнеризация (Docker)

    Создан Dockerfile для упаковки Java-приложения в контейнер.

    Написан файл docker-compose.yml для одновременного запуска приложения и базы данных.

    Настроена внутренняя сеть между контейнерами для их взаимодействия.

Технологический стек

    Язык: Java 17/25

    Фреймворк: Spring Boot 3

    База данных: PostgreSQL 15

    Инструменты: Docker, Docker Compose, Maven

    Фронтенд: HTML5, CSS3, JavaScript
